# Runtime Telemetry

This package monitors data freshness, engine reliability, performance, memory, and cache behavior.

- Tracks cache hits, misses, and freshness by source.
- Records engine latency, row counts, failures, and rolling health.
- Monitors process memory, iteration errors, and whole-day reliability.

Telemetry measures system health, not market edge or future profitability. Generated telemetry remains local.

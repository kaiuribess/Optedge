# Research Engines

This package collects and scores the independent market evidence used by Optedge.

- Fetches and normalizes public or locally configured data sources.
- Covers options, fundamentals, filings, sentiment, macro, technicals, market structure, futures, and catalysts.
- Converts evidence into standardized factors and degrades safely when a source is unavailable.

Optional credentials, caches, and collected results belong only in ignored local files.

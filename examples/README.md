# Examples

This directory provides small, sanitized examples of generated research artifacts.

- Shows the expected artifact structure.
- Supports documentation and schema regression checks.
- Keeps illustrative output separate from live research results.

Examples are not current performance evidence and must never include account-specific data. Live reports stay in ignored `data/` paths.

See the [complete project map](../docs/PROJECT_MAP.md) for the purpose of each tracked example.
